Rights

Rights are doled out by putting users into a rights group. different rights groups are represented by one alpha character.

    Rights Groups:
  • G - Anonymous ( special group for the unauthenticated users.
  • U - Unverified ( user signed up, user account must be changed to v to gain full user rights. )
  • v - verified ( usual user account, can search, request, kill, etc. )
  • m - moderator ( user + ability to modify otherr users requests in queue, kill user requests, etc. )
  • a - admin ( god )
