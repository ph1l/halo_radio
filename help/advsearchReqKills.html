Advanced Search Requests / Kills

This option allows you to specify what results you want listed.

  • If netrequests is selected then it will search for any songs matching query string that have a net request to kill ratio greater than 1 for the selected users.
  • If netkills is selected then it will search for any songs matching query string that have a net kill to request ratio greater than 1 for the selected users.
  • If requests is selected then it will search for any songs matching the query string that have been requested by the selected users.
  • If kills is selected then it will search for any songs matching the query string that have been killed by the selected users.
  • If both ignores songs that have a not been requested or killed by the users selected.
